# Data Professional Survey — Power BI Project

A Power BI dashboard analyzing survey responses from data professionals, covering roles, salaries, career switches, job satisfaction, and demographics.

## Project Overview

This project explores survey data collected from people working in data-related roles (Data Analyst, Data Scientist, Data Engineer, etc.) to uncover insights on:

- **Career paths** — whether respondents switched into data from another field, and how difficult the transition was
- **Compensation** — salary ranges across roles, industries, and countries
- **Job satisfaction** — happiness with salary, work/life balance, coworkers, management, upward mobility, and learning opportunities
- **Demographics** — age, gender, ethnicity, education level, and location of respondents
- **Programming languages** — most popular languages used by data professionals

## Tools Used

- **Power BI Desktop** — data modeling, DAX calculations, and report visualization
- **Excel** — initial data cleaning and preparation

## How to View

1. Download `Power_BI_Full_project.pbix`
2. Open it with Power BI Desktop (free): https://powerbi.microsoft.com/desktop/
3. Explore the interactive report pages and filters

## Key Questions Explored

- What percentage of data professionals switched careers into the field?
- How does salary vary by role, industry, and country?
- Which factors (salary, management, work/life balance, etc.) drive job satisfaction most?
- What are the most in-demand programming languages among data professionals?
